Place your custom audio files here and reference them in the app settings, for example:

- /public/sounds/default_alarm.mp3
- /public/sounds/default_tick.mp3

In the Settings modal, enter the relative path starting with `/sounds/`, e.g. `/sounds/default_alarm.mp3`.

Notes:
- For best compatibility use short MP3 or OGG clips.
- If the browser blocks autoplay, press Start (a user gesture) and then try preview.
